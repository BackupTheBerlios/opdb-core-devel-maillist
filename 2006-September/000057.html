Yes, it is available for download as an XML file which is used for
checking a site locally when using the toolbar in Offline Mode (the
default). In this mode it indeed downloads a local copy and checks for
updates in an interval which is configurable.
You can find the list of sites at http://opdb.berlios.de/opdbsites.xml.
