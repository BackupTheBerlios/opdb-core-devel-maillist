Hi,
Thanks for expressing your interest in FirePhish and the Open Phishing
Database project.
We currently don't have any plans for an non-plugin version of FirePhish,
which (statically) uses the
libphish<http://prdownload.berlios.de/opdb/libphish-0.1.0.tar.bz2>
anti-phishing
core library to interact with the database and manage the user's safe list.
The Konqueror Anti-Phishing Toolbar also uses this multi-platform core
library.

However, contributions to the project are welcome, especially if you can
write a pure JavaScript extension that uses the OPDB directly.

Regards,
Boaz Anin.
